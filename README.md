
# MiniAuth API 🔐

API de autenticación y control de acceso desarrollada con Node.js, Express y MongoDB Atlas.  
Incluye registro, login, JWT, protección de rutas y control de roles (admin/user).

---

## 🚀 Tecnologías usadas

- Node.js
- Express
- MongoDB Atlas
- Mongoose
- JSON Web Tokens (JWT)
- BcryptJS
- dotenv
- Thunder Client (para testing)
